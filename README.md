# Biblioteket

En app som visualiserer statistikk fra et Google sheet! Lagde dette prosjektet for å visualisere informasjonen på en fin måte. Leser veldig mye, og det er gøy å ha en oversikt over alle bøkene man har, derav Google sheet :D

## Python
Håndtering av lokalt regneark. Selve appen ligger i app/. Senere skal applikasjonen endres til å hente ut data direkte fra Google sheet ved å bruke en ID om det er mulig.
